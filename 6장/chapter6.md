## REST API
- 웹 자원을 HTTP 방식으로 요청하고 응답하는 API
- 주소와 메서드만 보고 요청 내용을 쉽게 파악할 수 있음
</br></br>

**규칙** 
- RESTful API는 동사를 쓰지 않는다  (예 : /articles/1 )
- 동사는 HTTP 메서드로 구현함 (예 : GET, POST, PUT, DELETE)
